This HTML file in this directory is used to running experimental tests.

It uses Electron in order to be able to read and write files to the
disk and run programs such as venneuler and eulerr.

To use it:
1) Install Electron (electronjs.org)
2) Open Electron and open batchTest.html
3) Select the directory of tests (contained in the testfiles folder)
4) When it runs, the results output HTML table will be shown and all 
   output will be written to the relevant testfiles folder.
